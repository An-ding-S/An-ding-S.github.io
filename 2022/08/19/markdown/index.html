markdown的简单语法

列表格式

列表格式也很常用，在 Markdown 中，你只需要在文字前面加上 - 就可以了，例如：

-文本1
-文本2
-文本3

如果你希望有序列表，
也可以在文字前面加上 1. 2. 3. 就可以了，例如：

  1. 文本
  2. 文本
  3. 文本

注：-、1. 和文本之间要保留一个字符的空格。

插入连接

语法： [显示文本] (连接地址)例如：

  • [小安的博客] (https://an-ding-s.github.io/)
  • 插入图片同上，只是前面多加了个！的符号。
  • 例如:![] (https://gcore.jsdelivr.net/gh/zsxwz/tuchuang2@latest/2022/08/19/67a1d77a85ead3e68ff1d9134990e53e.png)
  • 此图片为图床连接。

标题

在 Markdown 中，你只需要在文本前面加上 # 即可，你还可以增加二级标题、三级标题、四级标题、五级标题和六级标题，总共六级，只需要增加 # 即可，标题字号相应降低。例如：

#为1级|##为2级|###为3级| …..

  • 注意：# 后面要加空格

粗体和斜体

Markdown 的粗体和斜体也非常简单，用两个 * 包含一段文本就是粗体的语法，用一个 * 包含一段文本就是斜体的语法。例如：

  • *内容 *
  • ** 内容 **
  • 此时的*后面没有空格

代办清单To-do List

  • 代码命令：- ［］为未办事项或者项目。-［×］为已办项目或者事项。

  • 符号后面加空格。

  • 这个X是英文的x，并非符号。

  • 注意：文章中所有符号均为英文键盘下的符号。
